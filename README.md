# EduTrack Student Management Mobile App Prototype

EduTrack is a modern, clean, and professional student management mobile app UI prototype designed in Figma. The prototype uses a purple, white, and yellow accent color system with rounded Material Design-inspired cards, buttons, and navigation.

## Figma Prototype

[Open the EduTrack Figma Prototype](https://www.figma.com/proto/GcuXuHnjcI8ii4rx8JDCOc/EduTrack-Student-Management-Mobile-App?node-id=2-20&t=f3oNv66xgQfdv9cr-1)

## Screens Included

- Splash Screen
- Login Screen
- Dashboard
- Attendance Screen
- Assignments Screen
- Grades Screen
- Notifications Screen
- Profile Screen

## Prototype Features

- Mobile layout designed for a 390 x 844 iPhone frame
- Bottom navigation for Dashboard, Attendance, Assignments, Grades, and Profile
- Clickable prototype navigation between screens
- Clean visual hierarchy and consistent spacing
- Reusable component structure and organized layers
- Material Design-inspired cards, buttons, badges, input fields, and progress indicators

## Important

For the best viewing experience:

1. Open the Figma Prototype link.
2. Click the Display Settings icon in the top-right corner.
3. Select Fit width and height under Scaling Options.

This ensures the prototype is displayed correctly across different screen sizes.
